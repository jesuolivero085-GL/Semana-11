La plataforma tendría una interfaz sencilla y fácil de usar.

Incluiría un formulario para crear una nueva causa comunitaria con campos como título, descripción y fecha.

También mostraría una lista con todas las causas disponibles para que los usuarios puedan consultarlas.

Cada causa tendría un botón para apoyar la iniciativa.

Además, se mostraría un contador con el número de apoyos recibidos.

La interfaz no debe mostrar información privada de los usuarios y debe ser clara para facilitar su uso.
